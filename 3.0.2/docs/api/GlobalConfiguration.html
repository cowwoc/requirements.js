GlobalConfiguration

GlobalConfiguration

The global configuration inherited by all verifiers.

Note: Verifiers inherit from the global configuration at instantiation time. Their configuration is not affected by subsequent changes to the global configuration.

However, updating settings not found in Configuration (such as withTerminalEncoding(TerminalEncoding)) will impact the behavior of existing verifiers. By default, assertionsAreEnabled is false and isDiffEnabled is true.
