Interface InetAddressValidator

Validates the requirements of an IP address or hostname.

Verifier and Validator methods are equivalent. Validators return validation failures through the getFailures() method, while Verifiers throw them as exceptions.

All methods (except those found in ObjectValidator) imply isNotNull.
