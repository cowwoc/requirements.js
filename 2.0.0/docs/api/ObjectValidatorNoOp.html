ObjectValidatorNoOp

ObjectValidatorNoOp(failures) → {ObjectValidatorNoOp}

An implementation of ObjectValidator that does nothing. A validator that ignores all subsequent failures because they are guaranteed to fail and would add any value to the end-user. For example, an attempt was made to dereference null or cast the value to an incompatible type.
